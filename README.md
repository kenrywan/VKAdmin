# VKAdmin

VKAdmin - Free Admin Control Panel Themes Based on Bootstrap 5

-------------------

## Install

1. **Clone Repository**

```bash
git clone https://github.com/kenrywan/VKAdmin.git
cd VKAdmin
composer install
```
